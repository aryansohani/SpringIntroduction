# 🌱 Spring Boot Basics

This repository contains my **initial practice code while learning Spring Boot**.
It demonstrates how Spring manages objects using the **IoC Container**, along with basic concepts like **@Component**, **@Autowired**, and **ApplicationContext**.

---

## 🚀 Concepts Practiced

* Creating a **Spring Boot Application**
* Using **@Component Annotation**
* Understanding **Spring IoC Container**
* Retrieving beans using **ApplicationContext.getBean()**
* Using **@Autowired for Dependency Injection**
* Running a basic **Hello World program in Spring Boot**
